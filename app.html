Apple Beta
  • How to get Developer Beta?

    •Sign in at Apple Developer

    •Go to Settings>Privacy & Security and enable Developer Mode

    •Go to Settings>General>Software update

    •Select Developer Beta

    Note: Before installing any beta it is recommended to do a backup in case something goes wrong



  • How to get Public Beta?

    •Go to Settings>General>Software Update

    •Choose Public Beta

    Note: Before installing any beta it is recommended to do a backup in case something goes wrong
